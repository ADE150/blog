      # 认知能力

      # 简介

      本篇文章主要总结公众号「富研社」的如何改造思维，提升认知能力。目前作者该部分内容完全免费，但内容精炼易懂，层层递进，满满干货。

      我于21年9月花了一个月的时间研读了其32篇文章，受益匪浅。如今自己有足够能力，重新消化一下去年做的笔记，同时秉持开源精神，将干货无偿分享给大家。

      大家有兴趣可以关注「富研社」查看原文！

      合法地赚钱，只有三种方式：靠资源优势，靠信息差，靠机遇。

      认知能力，就是通过细微的观察和练习，发现别人发现不到的，从而制造信息差和提高机遇可能。

      要想真正发挥认知能力的效果，重要的是认知升级：

      • 升级“硬件”——概念
      • 升级“操作系统”——思维
      • 升级「操作员」——自我能力，包括任务管理能力，时间管理能力，执行力和团队协作能力
      • 安装「正版」——正确的方法，符合时代，较为新颖，符合你

      认知课程结构：概念重塑、脑力提升、效率升级、财富创造

      必不可少的三要素：学习，思考，实践

      # 课程大纲

      # 01 认识注意力——人生最宝贵的财富

      交换，是财富的基础。

      交换都是等价的，而风险也算是成本的一种，所以去偷去抢也需要花费大量的成本。

      人最宝贵的财富是注意力。金钱只是交换的形式，并非自己拥有。时间无法左右，身体是消耗品，不会增值。大脑不会思考是没有价值的，真正交换出去的并不是大脑，而是注意力。

      # 注意力的重要性

      注意力使我们能够影响身边物质的重要因素，它本身是一种能量。

      当你的注意力关注到什么的时候，就是在给它输送能量，你就是在改变着它。

      # 注意力的消耗形式

      你不珍惜注意力，总会有人替你珍惜注意力。

      # 1. 浪费掉

      关注那些无法被影响的事，能量射向天空

      # 2. 被收割

      各种通知让你应接不暇，能量被打包卖给广告商

      # 3. 被利用

      利用大数据分析，个性化收割注意力

      # 注意力的正确使用方式

      • 提升人际关系，尤其是亲密关系
      • 关注新趋势，以免成为井底之蛙
      • 聚焦当下，能产生实际价值的东西
      • 自己的成长，提高自己的能量密度，提高未来资源的效率性

      # 如何成为一个强大的人

      • 看一本书，丰富知识
      • 写篇日记，总结得失
      • 写篇分享，发射能量

      注意力能量密度的增长是符合指数增长的，哪怕每天增长1%，一年后的自己会比现在的自己厉害37.8倍。

      # 如何才能实现财富自由呢？

      用你全部的注意力交换！

      # 总结/思考

      • 你的注意力都去哪了？
      • 你的注意力可以用在什么地方？

      # 02 认识时间——世界上唯一的商品

      产品的本质是交换，我们能够交换的只有时间创造出来的价值。

      看似比的是“价值”，其实比的是“模式”：零售？批发？

      # 时间与价值的核心公式

      个人生产总值=你创造的价值/单位时间*用于创造价值的时间

      从公式出发，可以得到时间商人的四种经营模式

      # 零售时间

      把时间单份售出，核心是如何提高时间单价。

      # 批发时间

      把同一份时间卖给尽可能多的人，要学会善用互联网技术把边际成本降为零。

      # 买卖时间

      本质是个放大器，通过买入别人的时间，来提高自己的效率、提升时间单价、扩大生产规模。

      # 收时间税

      建立一个平台，让尽可能多的人在上面出售自己的时间，通过收税来赚钱。平台不是原因而是结果，从单点出发，先想清楚你能为谁赋能。

      赋能内容：

      • 提高他单位时间内创造的价值
      • 提高他的时间批发量

      # 买卖时间的条件

      满足两个条件之一，你可以将你的时间变现售出：

      • 创造价值：你在某方面更专业
      • 提升效率：你可以成为他的帮手

      所以，在团队中，要想发挥协作最大价值，要做到：

      • 把适合的人放在适合的位置上，可以发挥时间的最大创造能力
      • 前期团队需要创造价值的人，后期扩张复制需要提升效率的人

      # 03 设计人生——普通人如何逆袭

      现实：拥有资源优势的人仍然处在优势，劣势的人仍然处在劣势。

      但是命运也有“漏网之鱼”

      漏网之鱼的特点：

      • 能够坚持
      • 坚持的是内心喜欢的事

      一个人只能从优秀走向更优秀，所以要想逆袭，需要寻找自己身上的发光点和兴趣点，而不是到处碰壁磨平棱角成为一个普通人

      # 人生密码

      # 1. 找到自己的天赋

      • 显性天赋：长得高，长的帅，唱歌好听……
      • 隐性天赋:当你做某件事的时候，它能给你带来持续的愉悦感和满足感

      # 2. 重新设计自己的人生

      • 设定目标，并赋予伟大的意义:——设定“身份”，赋予“使命”(身份是你想成为怎样的人，“使命”指如何“利他”)
      • 思考商业化，把天赋变成事业——(商业化具体看后两个季度的内容)
      • 刻意练习“套路”，完成一万小时的积累——学习前人总结的经验
      • 设立一个个小目标，逐个完成它们——通过形成经验给自己一个正反馈

      # 04 复利效应——知者赚，不知者被赚

      世界上的事物，都有着共同规律——呈现爆炸式增长

      事物发展趋势图：

      # 复利的本质

      当你做了事情A，就会导致结果B，而结果B又会加强A，如此不断循环，循环次数越多，A就越强大。其实就那么简单的一个逻辑。

      # 复利的方法

      # 1. 找到必然关系

      找到一个因果关系，最快的寻找方法就是看书，站在巨人的肩膀上

      因果关系可以是公式，只要有因果关系就行

      # 2. 设计增强循环

      重在结果以资源的形式作用于条件，如：

      • 天生就有增强循环:利滚利，网约车……
      • 需要补充要素的增强循环:线下门店扩张

      # 3. 重复与耐心

      不断坚持，在“转折点”(里程碑)以下的努力是为了之后的增长做铺垫，初期很漫长。

      # 两个特殊的复利模型

      # 1. 加法运算→加幂运算

      加法运算：知识以点的形式增加

      加幂运算：知识以联结的形式增加

      # 2. 量变→质变

      影响中间增长率的核心要素，就是你复利的质量。

      我们所遇到的一切逆境，所感受到的一切委屈，以及正在经历的一切不开心，其实都是我尚处在地平线之下所致。

      # 阶段总结：坚持复利=复利增长=财富自由

      # 财富增长的底层密码：复利

      世界第八大奇迹，知者赚不知者被赚。我们懂得了这个概念，却并不知道该如何使用。现在你已经学会了：第一，找到必然关系；第二，设计增强闭环；第三，重复与耐心；

      # 产生复利效应：你的天赋

      如何才能屌丝逆袭？在这个寒门再难出贵子的时代，光努力已经不够了，你需要牢牢把握住每个人与生俱来的那仅有的1%的优势，让它成为你逆袭的重要法宝，产生复利效应。

      # 天赋商业化的方式：做一个时间商人

      我们在这个世界上只经营一种产品，就是时间，人生财富的高低，就在于经营它的不同方式，你是在零售时间？批发时间？买卖时间？还是在抽时间税？

      # 实现一切的方式：注意力

      所有的财富都是通过交换得来，而注意力是你唯一拥有的最宝贵的财富，它决定你的行为，决定你的思考，决定你的成长，决定你未来的财富，你不珍惜它，自然会有人替你珍惜。

      # 05 去角色化——脱了衣服你还是谁

      每个人在生活里扮演多个角色，每个角色都有不同的行为准则和处事态度

      # 角色化的作用

      • 便于管理和协作
      • 让你的行为变得可以预期

      # 角色化的方法

      教育

      # 角色化的弊端

      • 产生恐惧
      • 被束缚
      • 忘记「真实」

      # 去角色化的意义

      亲密相处，因为其本质是去角色化地交流

      # 去角色化的方法

      # 认识个人层面

      # 第一层:感知层

      在这层，能够感知一个人的外观性格，即“第一印象”

      # 第二层:角色层

      在这层，所做的动作，都是被这个角色所影响，甚至是被控制的。 交往到这层的人，只能是浅层次的关系

      # 第三层、第四层:资源结构层，能力圈

      在这层，资源结构层是指你的财富资源、人脉资源、精神资源、等等… 能力圈是指你的各种专业技能、管理能力、商业能力、沟通能力、等等… 角色会改变，而资源结构层和能力圈只是不断的进化和扩张。

      # 第五层:存在感知层

      在这层，你会感觉自己希望在哪有存在感:被点赞，被注意，还是创造伟大事迹的被铭记？

      你希望自己是一个怎么样的存在，就能驱动着你能走多远，你的边界就会被扩展到哪里。到这层，是一个人的内核。

      # 利用去角色化建立人际联系

      你要想和一个优秀的人在一起，并和他建立深度关系，就既要懂他真实的快乐，更要懂他真实的痛苦，并且给予他能量，支持他的存在感，互相依赖共存，才能成为灵魂互相交织的伙伴或者伴侣。

      # 利用去角色化看到人生可能性

      被动的人生是从外往内探寻的：很多人被困在角色层，在角色下生活，一生都在寻找人生的意义；

      主动的人生是从内往外拓展的：是去角色化的，一生都是在自我成长，没有边界的限制纵情向前；

      作者小结

      为什么今天我们要学习角色化？ 第一，为了理解自己：角色意味着规则、边界和束缚，有太多的条条框框，让你无法突破自己。你在什么时候，应该做什么，不应该做什么，其实都被确定好了，你是没有自主权的。你以为是你在思考，其实你是在替角色思考，你想做的事情，其实是角色想让你做的事情。你想要突破这些限制，你就要看到自己的存在感知层，从这里出发，扩展你的能力圈和资源结构层，你才能突破角色的边界，纵情向前。

      第二，为了理解对方：如果你追求的是深度关系和长期关系，那你需要看到的就绝不仅仅是对方表面上已经呈现出来的结果。基于角色层的交流都是浅层次的，你需要来到对方的存在感知层，了解他真实的快乐，更了解他真实的痛苦，交换彼此的能量，才能成为灵魂的伙伴。

      第三，为了理解陌生用户：虽然我们还没有开始讲产品，但是今天的内容对你未来设计一款产品有巨大的影响，因为我们每个人在使用绝大多数产品的时候，都是在去角色化的状态下使用的（有些职场类的app是基于角色化的产品，通常都很反人性）。

      当一个用户「演」了一天的角色，回到家躺在沙发上，想「去角色化」的自己待会儿，脑袋放空的刷刷存在感，你凭什么要求她按你设定的人物角色，对你的app进行操作呢？

      比如你设计了一款母婴产品，你觉得对方是一位母亲，就应该喜欢看这个，应该喜欢用那个...凭什么？

      为什么她就不能玩一晚上的王者荣耀？刷刷五杀的存在感？和自己的萌宝刷个抖音短视频？

      你如何穿透角色，与用户的存在感相处，顺应人性的设计产品？这才是一个优秀产品经理应该具备的独特视角。具体的方法， 我们会在第四季度的课程里详细讲解。

      # 06 理解层次——你是第几流人才

      NLP理解层次(NLP:神经语言程序学)「NLP理解层次」说：

      对一件事情的理解，我们可以分成6个不同的层次，而这个层次是有高低之分的。为了方便理解，按其分为六种人提前说明：这里说的每提高一个层次，并不是说就不要下一个层次了，比如有了方法就不需要努力了，而是在原来的基础上，上升了一个思考层次。不然就会变成空中楼阁，纸上谈兵。

      人才属性表：

      人才等级 别名 所理解层次 典型思考模式 解释
      第五流 怨妇 环境 都是你们的错！ 他们通常将问题归结于环境，解决办法是改变环境
      第四流 行动派 行动 我还不不够努力 要解决问题，会从自身行为上找问题，思维会停留在「不够努力上」
      我们要认识到努力只是成功的必要条件，而非充分条件
      第三流 战术家 能力 方法总比问题多 这类人有非常强大的学习能力和应用能力，能把学习到的知识，转化为可操作的方法，进而改善效率，解决问题。
      「能力」这个理解层次，是我们「意识」能想到的最高层次了。
      再往上走，就要进入到我们的「潜意识」区域每一个选择，都意味着人生的不同走向，一旦选择错了问题，你那优秀的「能力」和「行动力」只会让你越走越远。
      第二流 战略家 BVR：
      信念
      价值观

      第一流 觉醒者 身份 因为我是XXX，所以我会XXX 你之所以有时候会不知道该如何选择，除了对某些概念不清楚之外，最重要的就是你不知道自己想成为怎么样的一个人。
      角色是被动的，是别人给你的；身份是主动的，是你自己想成为的。
      顶级 领袖/伟人 精神/使命 人活着就是为了改变世界 顶级人才的行为和思考特点都是“利他”，并且有很强动力去试图改变世界，为世界做点什么

      # 07 元认知——你思考过你的思考吗

      元认知，是美国心理学家J.H.Flavell在1976年提出的概念，意思是：“反映或调节认知活动的任一方面的知识或认知活动，即认知的认知”。

      简单来说，元认知，就是你“对自己思考过程的认知与理解。”

      # 08 稀缺性/多维竞争——你值多少钱

      需求决定产品是否有价值

      场景决定产品是什么价值

      需求的另外一面是供给

      # 价值的决定因素

      需求：需求决定有没有价值，场景决定是什么需求；

      供给：供给量决定价值的多少，越稀缺的东西越有价值。

      你的价值不是和你的劳动成正比，而是和你劳动的不可替代性成正比

      # 打造稀缺性的方法

      # 1. 成为第一

      条件：抢占先机，获得优势

      适用性：比较难

      # 2. 成为唯一

      条件：拥有多维相关的能力

      适用性：好

      # 如何成为唯一

      利用多维能力——把多个能力结合在一起，能彼此互补借力，为同一个目标服务，这样就会发挥巨大的价值！

      # 步骤

      1. 先把一个能力打造成自己的长板
      2. 让自己兴趣广泛
      3. 确定一个目标，并把多维能力组合起来

      # 多维能力发挥价值的两个前提

      1. 每个能力都有价值，都是别人需要的
      2. 每个能力都要有关联，共同服务于一个大目标：多才多艺≠多维能力

      # 为什么要打造多维能力？

      # 如何打造自己的稀缺性

      你平时要多学点跨界的知识和技能，为打造多维能力做好基础，一旦目标出现后，你所学的哪些知识就会自动的组合成一个整体，让你成为稀缺型人才，让你变得不可替代，让你变得价值连城！

      最后，再次强调人生目标与使命的重要性，没有这个，你的兴趣广泛最终也只会变成你的多才多艺…

      # 09 趋势——可以预见未来，帮助我们决策

      # 利用趋势的两种方式

      • 外力
      • 势能差

      # 外力：突发事件（新闻，灾害，热点）

      存在的问题：

      • 不知道什么时候来，这个很难预测，只能等
      • 即使来了，你也不知道它未来究竟会如何发展，往往重大事件之后还有重大事件，也许一开始看上去是好事，但是结果却越来越糟糕。

      # 势能差

      找到有趋势差的地方，然后站在高势能的这边，跟着他一起往低势能处流动。

      # 势能差的重要性

      • 效率势能：优打劣，快打慢，廉打贵
      • 规模势能：大打小《孙子兵法》的道理，都是技巧一般的勇士，用以强胜弱的方式打败对方，以弱胜强的故事都是幸存者偏差
      • 认知势能：颠覆式创新
      • 引用势能：通过平台赋创造吸引力。就是某个事物，能够像拥有引力一样的吸引周围的事物，让他们成为自己的一部分的能力，类似一个黑洞一样的存在。比如:城市化，新的经济体

      # 势能差的应用例子

      现在的大企业学会了2个新的方式来应对这种「创新者的窘境」

      1. 要么从内部独立出去一个团队，脱离原公司，在创业环境下单独搞新业务
      2. 要么看到好的公司直接买买买。

      总之，不让他们在企业内部和老部队直面交战。

      # 站在势能高点的方法

      # 成为：让自己成为高势能的一部分

      去一家有势能的公司，如果是小公司，就需要利用好效率势能，关键是迅速，实现单点突破;还要利用好认知势能，实现高维打击，不过这种机会不多

      # 赋能：找到高势能的「线、面、体」来帮助自己提供势能

      附着于平台，平台可以给点提供技术支持与用户支持

      这里我借用阿里巴巴的曾鸣教授和湖畔大学的梁宁老师都提到过的一个概念叫做「点、线、面、体」战略分析框架来分析这个问题。

      • 点：可以指个人，或者某个单一产品；
      • 线：一个小公司，也可以指大公司里的一条业务线；
      • 面：指平台型公司，或者生态型企业，比如阿里巴巴、腾讯；
      • 体：指时代、行业、新的经济体；

      有了这个分析框架，你看自己的位置和看市场的竞争，就能有更立体的视角了：

      • 「点」自身唯一能把握住的是「认知势能」，其他基本都不具备；
      • 「线」可以拥有「效率势能」和「认知势能」，势能比「面」小一个数量级，比「点」高一个数量级。
      • 「规模势能」和「引力势能」是「面」和「体」独有的，而「体」的势能比「面」高出一个数量级；

      有了这个分析框架之后，我们就有了寻找、并站上势能至高点的方法了。

      # 作者总结

      从这一课开始，你除了要关注自身的成长，还要开始关注所处的环境。环境，是影响你人生走势的一个更大的变量，你绝对不能忽视它，有时候，人生的选择远比努力更重要，你要学会顺势而为。

      那么你如何找到未来的趋势呢？

      • 产生趋势的原因有两个：1. 外力；2. 势能差。但是外力的发生不可预测，未来的趋势捉摸不定，因此我更推荐“寻找势能差”这个策略。
      • 势能差一共分为四类：1.效率势能；2.规模势能；3. 认知势能；4.引力势能。
      • 想要抓住由“势能差”带来的趋势，你可以通过“点、线、面、体”这个分析框架，立体的看待所处的环境，找到适合自己的势能高点，或成为他们，或想办法让他们为你赋能。

      站在势能高点，也许你还是不知道未来的趋势会去向哪里，但是你却和趋势站在了一起，抓住未来世界给你留下的信号，让自己成为未来的一部分！

      # 10 估值模型——我有一个改变世界的想法，可行吗？

      # 估值维度

      • 估算客户终生价值：对「盈利能力」的估算。客户终生价值=（客单价-边际成本）×购买次数
      • 估算获客成本：对「发展能力」的估算。获客成本=市场投入÷获得流量÷付费转化率
      • 估算市场：对「成长空间」的估算。用户规模=市场规模×占有率
      • 估算风险成本：对「风控能力」的估算。存在于每个环节，应该想办法转移，或者通过2B业务支持2C业务，保有高成长的机会

      # 作者小结

      用这4个概念，你就可以对项目的「盈利能力」、「发展能力」、「成长空间」、「风控能力」进行全面的评估，这个项目到底值多少钱，值不值得你投入，投资人看不看得上…

      把这4点组合起来，你就能得到一个完整的估值模型：

      # 「盈利能力」是核心

      它考验你产品的质量，产品的价值，对用户需求的理解，对用户持续的服务，供应链的管理，对新技术的理解和应用…等等，这些所有的目的只有一个，提高客户终身价值。这是你的能力内核，是一切的核心与起点。

      # 「发展能力」是看你能把你的盈利能力以多快速度、多大规模的放大

      它包括了你的营销能力，渠道开发能力，品牌建设能力，内容运营能力…等等，这些所有的目的只有一个，降低获客成本。

      # 「市场空间」是你的发展极限，是你的天花板

      你在开始的时候就得想清楚，你想做多大的生意，规模和竞争是成正比的，你进入的赛道越宽，和你竞争的人就会越多，你的能力配得上多少市场份额？

      并不是赛道越宽越好，有时候小赛道，没什么竞争，你只要做了，就能成为前几名，就能霸占赛道，甚至关闭赛道，让后来者觉得没必要进来了，你也能获得不错的项目回报。

      # 最后一点是「风险」，它存在于你的每个环节，每个阶段

      但很多创业者经常对它视而不见。而风险永远不可能真正消除，你无法预料到它会以什么方式在什么时候来到你的身边，再大的公司都有倒闭的一天，比如诺基亚倒闭的时候说：“我们并没有做错什么，但不知道为什么，我们输了。”

      既然不可避免，那你就得先想好怎么将风险转移出去。

      比如，找投资人融资的本质，其实就是为了你的创业买了一份保险，当然，相应的你得拿出你未来公司成长带来的收益去交换；

      或者你可以通过资产配置的方式转移风险。

      比如，同时开启两项业务，一份是没有什么未来，但是能带来稳定现金流的2B业务，一份是初期没有什么收入，但是有很好成长性的2C业务，然后用2B业务的现金流来养2C业务，这样公司不至于现金流很快枯竭而死掉，还能保有高成长的机会...

      如果把这4项组合起来用公式表示，那就是：

      项目估值=(客户终生价值-获客成本)×用户规模-风险成本

      这并不是一个严格意义上的数学公式，因为少了参数，还记得我们第三课讲的一个概念吗？

      数学应该是一个思考工具，表达工具，而不是计算工具。

      我们应该要透过这个公式，来理解各个概念对你这个项目的影响关系，帮助你更客观的来评判自己的项目是否可行，如果沙盘上都没推演成功，实战中又怎么可能有奇迹发生？

      来，试着用它为你还没落地的想法，或者为你已经开始的项目估个值吧！

      等等，你是不是感觉这个模型里好像缺了什么？

      对，这个模型里还缺3个超级大变量：

      一个是上节课讲的趋势/势能；

      一个是我们还没有讲到的团队；

      还有一个就是你自己！

      如果把上面这个项目模型看成是一个球，那么你该如何撬动这个球呢？

      你是不是想到了阿基米德说过的话了？

      “给我一个支点，我可以翘起整个地球…！”

      这个支点是什么？

      支点就是你自己，你的能力够不够强，能不能作为一个稳定的支点？你能不能扛起杠杆的两端？还是个空心鸡蛋，一压就碎了？

      杠杆的另外一头站着什么？

      就是你的团队，你有一个想法，光有你自己这个支点是不行的，你得学会借助团队的力量，让他们团结一致，朝同一方向使力。

      团队靠人数多是没有用的，往下压的力量只有两个：创新和效率。

      那趋势/势能是什么？

      趋势/势能就是杠杆本身，你越是在趋势里，越站在高势能处，你的杠杆就越长，团队撬动你这个项目就会更容易。

      现在，你终于可以面对自己那灵光一闪的想法，做一个全面的分析了：

      • 这个项目的盈利能力如何？用户是不是真的需要我？客户的终生价值是多少？
      • 这个项目的发展能力如何？我有没有存量的资源可以用？能否找到降低获客成本的方式？
      • 这个项目的市场空间如何？是个没人竞争的小赛道？还是有想象力的大赛道？
      • 这个项目如果赔钱了咋办？我能不能承受得起这个损失？是否可以融资来为风险买个保险？
      • 你自己有没有准备好？有没有相关的经验？商业知识够不够？
      • 这个项目是否在目前的趋势里？有哪些高势能的平台能为我的团队附能？
      • 你团队成员都有哪些人？他们的战斗力如何？是否能够齐心协力？是否拥有「创新」和「效率」这两把武器？

      感觉把这些问题都能认真的回答一篇，都赶得上写一篇作文了… 对！

      很好，有问题，而且问题都已经变的非常具体了，这已经比盲目开始要强的多了。
