        # 学习方法总结

        2022-05-19 20:59

        # 提高学习效果的方面

        从以下方面提高学习效果：

        • 学习心态
        • 学习动机
        • 学习状态
        • 学习方法

        # -01- 学习心态

        以下原则贯穿学习始终，甚至整个人生行动：

        • 行动很重要, 千里之行始于足下。
        • 有人指路和没人指路差别很大。
        • 阶段性目标。
        • 获得正反馈。
        • 圈子很重要。
        • 战术上重视，战略上藐视。这绝对不是一句空话。
        • 往往会逃避的地方，都值得思考是否有价值。
        • 学习起步是痛苦的。
        • 相信自己，不要自我设限。
        • 仅放目的的过程性迷失。
        • 学习是在起起落落中进步，偶尔堕落也是正常的，人不是机器。
        • 学习是没有立竿见影的效果
        • 做好终身学习的准备，把学习当成日常。

        # -02- 学习动机

        学习动机可分为明确的目标带来的效果，或者人为地制定奖励。人为奖励无法持续供应学习动机，最重要的还是找到自身的内在动机，即能否成就自身某个价值，能否“利他”。

        一个有效的动机是，你在想到它的时候，能否迅速感到自己充满力量，从而去行动。真正有效的动机需要的意志力是很少的，这类动机通常是初衷，具象化的未来预期和应用性的东西，又或是对他人有利的（有利于他人的总是不乏他人的赞美，鼓励和帮助，这些东西很有积极意义）。

        对于我来说，我的动机是：

        成为一个干净卫生，阳光有趣的大男孩，无论碰见感情问题，社交尴尬，我都能通过自己高效的搜索，有趣的表达和强烈的共情能力，倾听以分担痛苦，微笑幽默以走进人心。

        成为一个极度自律，原则明确的人。在职场上有职场规则，与人相处有相处规则。有自己的选择，而且自己知道自己能干成一番大事。

        不拘泥于现在的视野，疯狂寻找信息弥补自己的不足。每天不拘泥于自己现在的能力，时刻明白自己想干的大事是什么。多看看别人是如何做的，探索其中成功的本质和失败的原因。用公共的数据支撑理论，在公共新闻中寻找自己的机会。

        # 找到自身目标

        多尝试，多复盘，从顶层思维(利他角度)寻找社会价值，从自身擅长挖掘自身价值。

        多听他人指导，如果没有条件，就多看书，都是精华。

        迷茫的时候，去读人物传记。

        看看环境或者记忆内是否有你希望成为的榜样。

        # 明确目标

        SMART原则：

        • 绩效指标必须是具体的（Specific）
        • 绩效指标必须是可以衡量的（Measurable）
        • 绩效指标必须是可以达到的（Attainable）
        • 绩效指标是要与其他目标具有一定的相关性(Relevant)
        • 绩效指标必须具有明确的截止期限（Time-bound）

        # 在规划中落实目标

        将目标分为需要完成的任务，细分为可以可以轻松完成的习惯，或者稍微有挑战性的任务，然后把主要任务安排在每天效率较高的时刻中。

        有的放矢地完成任务，如使用二八原则，选取20%核心任务完成，因为这20%直接决定了你项目80%的质量。

        # -03- 学习状态

        # 排除所有的干扰项

        # 排除干扰

        拒绝所有的无用社交。只要面对的是人，你无法控制对方的话题，亦或让你兴奋，亦或让你焦虑，都会影响学习。最好的方法就是少见人遇事，一是心不静，二是耽误时间计划，影响计划，陷入焦虑。

        戒掉手机。能用电脑用电脑，学习时要开“静音”+“不振动”

        找一个适合自己的环境学习。保证尽量安静，如图书馆。乱不乱看自己平时高效的环境是否乱，如果乱，那就保持在乱的环境学习，不必刻意去整理。

        学习时，远离原本分配给“舒适氛围”的环境，如床，寝室。

        远离诱惑，尤其是实时更新的，如股票基金……

        # 适应干扰

        毛泽东读书，总会去喧闹的人群，或在极为恶劣的环境下读书，这锻炼了他的意志力和适应能力。

        如果对学习效果不是很急，可以先尝试拿着学习资料，去吵闹的地方读书，去锻炼屏蔽干扰的能力。

        # 快速进入学习状态的方法

        身体先行。

        立刻开始学习。

        从简单的开始，状态好可以从难的先开始。

        制造时限和压力，最大化认知效率。

        锻炼学习专有的仪式感。十几秒，非学习不用的那种。

        选择适合的学习环境：光线，温度，通风，远离舒适区。

        # 保持学习效率的方法

        优先保证完成学习计划。

        交叉学习，穿插安排自己感兴趣或擅长的。交叉科目/章节/行动学习，让大脑在左右脑，或者不同区域间切换，缓解大脑疲劳。

        保证学习开始和末尾不要做太难的内容，切忌在痛苦中结束学习。原理是峰终定律：如果在一段体验的高峰和结尾，体验是愉悦的，那么对整个体验的感受就是愉悦的。

        在睡眠，饮食，运动中下功夫：

        • 适量且适合自己的有氧运动
        • 充足的睡眠
        • 午睡：0 < Time < 30min。睡不着就冥想放松。
        • 清淡饮食，少吃辛辣油腻。饭吃七分饱较舒服且消化负担小。

        # 其他方法

        听轻音乐，声音不要太大。

        适量吃水果和小零食。

        定闹钟闭目养神/冥想20分钟。

        学习时停止一些小习惯。

        头脑不清可以洗漱。所以睡前1小时洗漱是个很好的选择。经常去图书馆的，一次性面巾纸是个不错的选择。

        # -04- 学习方法

        # 预习

        事先把要学习的科目的目录，大纲浏览一遍，如果有条件，可以简单浏览整篇教材讲了什么(留意特殊字体的字段)。

        建立树干比增添叶子重要，叶子多一片少一片并不会影响整体观感，但是枝干少一条会很明显。枝繁了才能叶茂。

        挑选一本好的教辅。好的排版和内容消化很重要。

        # 理解

        # 理解的原理

        理解就是寻找知识点中的某种逻辑。

        # 理解的技巧

        善于提取关键信息，不求甚解：

        • 找出作者的核心观点，即知识点(考点)
        • 提取知识点中的关键字/词，通过自己的理解，串联形成自己的逻辑思路，并做好笔记
        • 过关型考试大胆舍弃，不求甚解。这不是放弃，是战略。
        • 不要奢求一遍能过，循序渐进和重复是最高效的。
        • 多向自己提问
        • 多对比总结

        要及时做笔记。笔记方法遵循以下原则：

        • 根据自己的情况做笔记，记易混淆，容易忘的知识点
        • 不要为了拘泥于结构而把一些琐碎的知识点丢弃
        • 以少记多

        要及时记忆关键信息，把关键信息记住，内化为自己的经验后，再记其他内容就容易得多。

        先有枝干再有叶。在框架里填充，比强行在黑暗中摸索搭建要轻松且有效很多。

        # 练习

        做配套的练习。一是巩固知识点，二是了解命题方向，三是给自己正向反馈(证明自己学习是有效果的)。

        # 复习

        可以根据艾宾浩斯遗忘曲线回顾内容。

        每日用30分钟回顾今天学了什么。回顾形式的优先级如下：

        做一个错题本，记：01错题，02经典好题，03知识点的归纳和总结。要求题干完全，解析主观，每次做题像做新题一样对待(没有答案)，要对连续四次才划掉该题，放到一个文档实时更新。

        自己梳理一个框架，可以是画思维导图，也可容易使逻辑树等形式，重在过程(帮助理解)。

        做一个会做笔记的人。学会为自己的知识框架做笔记，同时多用简写，提高效率。

        # 真题

        (没啥好说的)要做成套真题，身临其境，体验时间感和难度，体验学习内容中的重点。

        # -05- 其他方法

        # 其他理论

        # 压力学习法

        创造适当的压力，如有挑战性的时间限制，DDL等

        # 峰终定律

        在主要的学习前后安排自己擅长或者感兴趣的学习内容或其他事情，要求做起来简单，不耽误事，该方法可以创造成就感，并留下美好的学习体验，形成学习正反馈。

        # 坚持法

        当感觉能量耗尽时，提供心理暗示：“已经做这么多了，再坚持一会吧”。

        Warning / 注意

        此方法只能在符合情况下使用1~2次，因为大脑是需要休息的，不能无限制榨干，否则适得其反。
